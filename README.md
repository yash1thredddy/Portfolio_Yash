### **About The Project** üí°

Static portfolios are a thing of the past. They present a one-sided, lifeless view of an engineer's journey. This AI-native portfolio breaks that mold.

This project is an **AI-powered, interactive portfolio** that showcases an E-shaped engineer's journey across multiple technical domains. Instead of forcing recruiters and developers to scroll through static content, it invites them to have an intelligent conversation about my experience, projects, and expertise.  

It's a portfolio that doesn't just show‚Äîit demonstrates depth and breadth through interaction.

### **Installation & Usage** üöÄ

Want to get this project running on your local machine? Follow these simple steps.

#### **Prerequisites**
-   Node.js (v18 or higher)
-   pnpm package manager
-   DeepSeek API Token
-   GitHub API Token (optional, for star count)

#### **Local Setup**

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/yash1thredddy/ai-native-portfolio.git
    cd ai-native-portfolio
    ```

2.  **Install dependencies:**
    ```sh
    pnpm install
    ```

3.  **Set up your environment variables:**
    Create a `.env.local` file in the root of your project and add the following keys:
    ```env
    DEEPSEEK_API_KEY="your_deepseek_api_key_here"
    GITHUB_TOKEN="your_github_token_here"
    ```
    *   Get your DeepSeek API Key from [platform.deepseek.com](https://platform.deepseek.com).
    *   Generate your GitHub Token at [github.com/settings/tokens](https://github.com/settings/personal-access-tokens).

4.  **Run the development server:**
    ```sh
    pnpm dev
    ```

5.  **Open your browser** and navigate to `http://localhost:3000`.
